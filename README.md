# Torneo Calcetto Capodacqua

Sito statico pronto per GitHub Pages.

## Pubblicazione
1. Crea un repository GitHub.
2. Carica tutti i file del progetto.
3. Vai su Settings > Pages.
4. Source: Deploy from branch.
5. Branch: main, cartella /root.

## Aggiornare risultati
Apri `/admin/`, password demo `admin123`, modifica il JSON e scarica `matches.json`. Sostituiscilo in `/data/matches.json` e fai commit.

## Future edizioni
