# Torneo Calcetto Capodacqua

Versione definitiva ampliata.

## File principali
- `index.html`: home ufficiale.
- `teams-2025.html`: squadre della 3ª Edizione.
- `matches-2025.html`: partite e risultati 2025.
- `archive-2025.html`: archivio storico.
- `team.html?id=...`: pagina automatica squadra.
- `admin/index.html`: pannello statico per scaricare JSON aggiornati.
